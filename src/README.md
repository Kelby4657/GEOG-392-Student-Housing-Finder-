# Source Code Directory

This directory contains the main application code for the Student Housing Finder web app.

## Purpose

Store all source code files including:

- **Application Logic**: Core functionality for the housing finder
- **Web Components**: Frontend components and views
- **API Services**: Backend services for data processing
- **Utilities**: Helper functions and modules

## Structure

Organize code into subdirectories as the project grows:

```
src/
├── components/    # UI components
├── services/      # API and data services
├── utils/         # Utility functions
└── pages/         # Page templates
```

## Features to Implement

The application will provide:
- Search housing by location proximity to TAMU campus
- Filter by commute options (walking, biking, bus, car)
- Filter by price range
- Interactive map visualization
