# GEOG-392-Student-Housing-Finder-

A web application for finding student housing near Texas A&M University based on location proximity to campus, commute options, and price.

## Project Overview

This application helps students find housing by providing:
- **Location-based search**: Find housing near TAMU campus
- **Commute analysis**: Compare transportation options (walking, biking, bus, car)
- **Price filtering**: Search within your budget

## Project Structure

```
GEOG-392-Student-Housing-Finder-/
├── data/           # Main data files (housing listings, location data, commute info)
├── src/            # Application source code
├── config/         # Customization features and configuration files
└── README.md       # This file
```

### Directory Descriptions

- **`data/`**: Contains data files including housing listings, geographic data for TAMU campus, and commute route information. Supports CSV, GeoJSON, and JSON formats.

- **`src/`**: Contains the main application code including web components, API services, and utility functions.

- **`config/`**: Contains configuration files for customizing the application, including app settings, map configuration, and filter presets.

## Getting Started

1. Add your housing data files to the `data/` directory
2. Configure application settings in the `config/` directory
3. Develop application code in the `src/` directory
