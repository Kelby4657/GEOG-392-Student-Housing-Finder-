# GEOG-392-Student-Housing-Finder-

A web application for finding student housing near Texas A&M University based on location proximity to campus, commute options, and price.

## Project Overview

This application helps students find housing by providing:
- **Location-based search**: Find housing near TAMU campus
- **Commute analysis**: Compare transportation options (walking, biking, bus, car)
- **Price filtering**: Search within your budget

## Project Structure

```
GEOG-392-Student-Housing-Finder-/
├── Raw Data/           # Main data files (housing listings, location data, commute info)
├── Source Code/            # Application source code
├── Web App/         # Customization features and configuration files
└── README.md       # This file
```

### Directory Descriptions

- **`Raw Data/`**: Contains data files including housing listings, geographic data for TAMU campus, and commute route information. Supports CSV, GeoJSON, and JSON formats.

- **`Source Code/`**: Contains the main application code including web components, API services, and utility functions.

- **`Web App/`**: Contains configuration files for customizing the application, including app settings, map configuration, and filter presets.

## Getting Started

1. Add your housing data files to the ` Raw Data/` directory
2. Configure application settings in the `Web App/` directory
3. Develop application code in the `Source Code/` directory
